# ShipSwift

iOS 开发组件模板库和后端开发文档集合。

## 目录结构

- `ShipSwift/slPackage/` — 可复用 SwiftUI 组件
  - 动画组件（扫描动画、渐变、Rive 加载器等）
  - UI 组件（图表、Onboarding、步进器、Loading 等）
  - 聊天组件（语音识别、消息列表等）
  - 管理类和工具类
- `ShipSwift/Docs/` — 后端开发文档（CDK、Auth、Database、Lambda 等）
- `ShipSwift/View/` — 示例视图
